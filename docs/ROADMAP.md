# Roadmap

Mise_Deck should stay focused, fast, useful on the Cardputer, and a little weird in the best cyberdeck way.

## v1.0.0

First public release candidate.

- English firmware experience
- English and Portuguese GitHub documentation
- Recipe library on microSD
- Simple and composite recipes
- Recipe scaling by total weight
- Timer and converter
- Battery and sound controls
- Wi-Fi setup
- Local browser portal
- Mobile-friendly portal layout
- Offline QR recipe sharing
- About screen

## v1.1 ideas

- PT-BR firmware build
- Portuguese / English language option
- More polished GitHub screenshots and media
- Better mobile portal polish after more real-phone testing
- QR sharing improvements
- Import from compact QR/text payload

## v1.2 ideas

- Recipe search
- Better category management
- More refined recipe editor
- Export individual recipe backups from the Cardputer UI
- More recipe templates

## Hardware future ideas

- Scale module integration
- Thermometer module integration
- Better battery/charging behavior if hardware APIs allow it

## Philosophy

Mise_Deck should remain:

- offline-first;
- SD-card friendly;
- readable on a tiny screen;
- fast during real kitchen use;
- open and hackable;
- slightly strange, charming, and cyberdeck-flavored.
