## Filming tips

- Use bright, indirect light.
- Film the Cardputer screen close-up.
- Avoid screen reflections.
- Record one clean phone/browser shot.
- Use captions for each feature.
- Keep cuts short.
- Put the GitHub URL in the final frame.
- If possible, show the device near real ingredients or a notebook to make the kitchen use case obvious.
