# Mise_Deck v1.5.0

Mise_Deck v1.5.0 focuses on a cleaner recipe workflow, a better mobile portal experience, improved sharing, and aligned English / Portuguese-Brazilian builds.

## Highlights

- New mobile-friendly portal layout
- Improved recipe creation and editing in the browser portal
- Backup menu with TXT and ZIP export options
- Improved recipe sharing
- Better recipe overview on the Cardputer
- English and Portuguese/Brazilian builds aligned under the same `v1.5.0` version
- General UI polish and bug fixes

## Downloads

- `Mise_Deck_Cardputer_v1.5.0_EN.bin`
- `Mise_Deck_Cardputer_v1.5.0_PT-BR.bin`

## Notes

Mise_Deck stores recipes locally on the microSD card and serves the browser portal directly from the Cardputer on your local network.

See `docs/INSTALL.md` for flashing instructions.
